
# Wordle
🎉 Welcome to **Wordle**!

## Overview
 This project was created on April 1st, meant for learning the basics of python, the current version 
 simulates the wordle game logic quite well, however, it is not very polished and will continue to be improved.
 
## Attributes
- ✅ Functional game logic
- ✅ Robust command line operation
- ✅ Game settings saved via ndjson
- ✅ English word checks
- ✅ Multi-length word choices
- ✅ Highly customizable
- ⚠️ No GUI exists as of yet
- ⚠️ The web scraping function is fragile and can be improved.

## Compatibility
- Platforms:
  - ✅ Windows
  - ✅ macOS
  - ✅ Linux

- Python Versions:
  - ✅ Python 3.9+

## Requisites
- Python 3
- Internet connectivity
- Interactive Python shell

## Notes
- Future releases will add GUI, features, and optimizations.
